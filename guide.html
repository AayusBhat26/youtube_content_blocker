YouTube Content Blocker - Quick Start Guide

Welcome to YouTube Content Blocker! This guide will help you quickly set up and start using the extension to filter out unwanted content from your YouTube experience.

Basic Usage

  1. Blocking Keywords: Any video with a title containing your specified keywords will be hidden.
    • Click the extension icon in your browser toolbar
    • Enter a keyword in the "Enter keyword to block" field
    • Click "Add" or press Enter
  2. Blocking Creators: All videos from a specific YouTube channel will be hidden.
    • Click the extension icon in your browser toolbar
    • Enter a channel name in the "Enter channel name to block" field
    • Click "Add" or press Enter
  3. Enable/Disable: Use the toggle switch at the bottom of the popup to quickly enable or disable the extension.

Advanced Features

Context Menu Options

Right-click on videos or channels to quickly block them:

  • Block This Channel: Right-click on a channel name and select this option
  • Block Keyword From Title: Right-click on a video to block based on its title
  • Custom Block Keyword: Add a custom keyword from anywhere on YouTube

Advanced Settings

Access more options by clicking "Advanced Options" in the extension popup:

  • Blocking Mode: Choose between hiding videos completely, blurring them, or replacing with a placeholder
  • Keyword Matching: Select partial matching, exact matching, or whole word matching
  • Case Sensitivity: Choose whether capitalization matters when matching keywords
  • Performance Settings: Adjust scan intervals for optimal performance

Import/Export Settings

You can back up your settings or transfer them to another computer:

  • Go to Advanced Options
  • Use the Export Settings button to save your configuration
  • Use the Import Settings button to load a previously saved configuration
Tip: Combining multiple keywords and channel blocks can dramatically improve your YouTube experience. Start with a few key terms and gradually build your filter list.
Note: If a video is blocked on the watch page, you'll see an overlay with an option to go back or view the video anyway.

Troubleshooting

If videos aren't being blocked as expected:

  1. Make sure the extension is enabled (check the toggle switch)
  2. Try refreshing the YouTube page
  3. Check that your keywords don't have typos or extra spaces
  4. For channels, make sure you're using the exact channel name as shown on YouTube
  5. In Advanced Options, try using "Partial match" mode for more flexible keyword matching

If you encounter any bugs or have feature requests, please visit our GitHub repository to report them.
